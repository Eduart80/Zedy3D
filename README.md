# Zedy3d — Portfolio & Freelance Website

Personal portfolio and freelance services website built with React and Vite. Showcases projects, skills, services, and pricing — deployed via GitHub Pages.

## Sections

| Section | Description |
|---|---|
| Hero | Introduction, name, role, and CTA buttons |
| Skills | Tech stack chips — languages, frameworks, tools |
| Projects | Featured project cards with tags, GitHub links, and live demos |
| Also Built | Compact list of additional work across the portfolio |
| Services | What I offer — Websites, E-Commerce, Apps, Maintenance |
| Pricing | Five tiers: Website · E-Commerce · App · Platform · Maintenance |
| Contact | Email, GitHub, and LinkedIn links |

## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx       # Fixed nav with mobile hamburger menu
│   ├── Hero.jsx         # Landing section
│   ├── Skills.jsx       # Tech stack grid
│   ├── Projects.jsx     # Featured projects + Also Built list
│   ├── Services.jsx     # Service offering cards
│   ├── Pricing.jsx      # Pricing cards + Maintenance
│   └── Contact.jsx      # Contact links
├── App.jsx
├── main.jsx
└── index.css            # All styles + responsive breakpoints
```
